   2) 프로그램 언어 ( 자바 )
      - JDBC ( Java DataBase Connectivity ) 기술
      - JDBC 특징
         DB 종류와 무관하게 연동 가능( DB에 독립적)
	 컴파일 checked 예외가 발생된다. ( 반드시 try~catch 또는 throws 해야 됨 )
